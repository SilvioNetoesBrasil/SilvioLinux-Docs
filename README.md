# SilvioLinux

**SilvioLinux** é uma distribuição Linux leve, moderna e minimalista criada por **Sílvio Ramalho Neto da Silva**.

O objetivo do projeto é oferecer um sistema operacional simples, eficiente e portátil, baseado inicialmente em distribuições tradicionais e posteriormente migrado para uma base construída com **Yocto Project**.

## Características

- Distribuição Linux leve
- Suporte a arquiteturas modernas e antigas
- Baseada inicialmente em Linux Mint
- Posteriormente migrada para Yocto Project
- Uso do BusyBox para sistemas minimalistas
- Sistema de pacotes opkg
- Interface gráfica GNOME ou XFCE dependendo da versão

## Arquiteturas Suportadas

- x86 32-bits
- x86 64-bits
- RISC-V 32 (RV32)
- RISC-V 64 (RV64)

## Componentes Principais

- Kernel Linux
- BusyBox
- Sistema de pacotes opkg
- Interface gráfica GNOME

## Objetivo

Criar um sistema Linux:

- leve
- moderno
- portátil
- educacional
- fácil de modificar
