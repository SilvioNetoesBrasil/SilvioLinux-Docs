SilvioLinux

SilvioLinux é uma distribuição Linux leve e minimalista criada por
Sílvio Ramalho Neto da Silva.

Objetivo:
Criar um sistema Linux simples, portátil e eficiente.

Características principais:

- Base inicial: Linux Mint
- Base atual: Yocto Project
- BusyBox
- Sistema de pacotes: opkg
- Interface gráfica: GNOME ou XFCE

Arquiteturas suportadas:

x86 32 bits
x86 64 bits
RISC-V 32 (RV32)
RISC-V 64 (RV64)
