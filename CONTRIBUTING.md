# Contribuindo para o SilvioLinux

Obrigado por considerar contribuir com o projeto SilvioLinux.

## Como contribuir

1. Faça um Fork do repositório
2. Crie uma nova branch
3. Faça suas alterações
4. Envie um Pull Request

## Regras

- Use código limpo
- Documente suas alterações
- Não envie código malicioso
- Teste antes de enviar

## Tipos de contribuição

- Correção de bugs
- Melhorias de código
- Documentação
- Scripts de build
- Novos pacotes
