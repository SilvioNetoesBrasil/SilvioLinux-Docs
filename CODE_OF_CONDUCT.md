# Código de Conduta

Este projeto segue um código de conduta para garantir um ambiente saudável.

## Regras

- Seja respeitoso
- Não use linguagem ofensiva
- Não assedie outros participantes
- Respeite opiniões diferentes

## Comportamentos aceitáveis

- Ajudar outros desenvolvedores
- Compartilhar conhecimento
- Colaborar de forma educada

## Aplicação

Qualquer violação pode resultar em remoção do projeto.
