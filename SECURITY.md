# Segurança do SilvioLinux

Se você encontrar uma vulnerabilidade de segurança no SilvioLinux:

## Reportar vulnerabilidades

Envie um relatório contendo:

- descrição do problema
- passos para reproduzir
- impacto potencial

## Política

Problemas de segurança serão corrigidos o mais rápido possível.

## Atualizações de segurança

Correções serão distribuídas através do sistema de pacotes opkg.
