# Instalação do SilvioLinux

## Requisitos mínimos 32-bit

- CPU x86 ou RISC-V
- 2GB RAM
- 25GB de armazenamento

## Requisitos mínimos 32-bit

- CPU x86 ou RISC-V
- 4GB RAM
- 25GB de armazenamento

## Requisitos mínimos 64-bit

- CPU x86 ou RISC-V
- 8GB RAM
- 80GB de armazenamento

## Requisitos mínimos 64-bit

- CPU x86 ou RISC-V
- 16GB RAM
- 80GB de armazenamento

## Instalação

1. Baixe a imagem ISO
2. Grave em um pendrive

Linux:

sudo dd if=silviolinux32-1.0.iso of=/dev/sdX bs=4M ou sudo dd if=silviolinux64-1.0.iso of=/dev/sdX bs=4M

sudo dd if=silviolinuxrv32-1.0.iso of=/dev/sdX bs=4M ou sudo dd if=silviolinuxrv64-1.0.iso of=/dev/sdX bs=4M

3. Inicialize o computador pelo pendrive
4. Execute o instalador
5. Siga as instruções na tela
